Himalayan Farm Fresh+

An AI-powered e-commerce platform designed to integrate Machine Learning with agriculture and online product management.

Overview
Himalayan Farm Fresh+ is a smart platform that combines agriculture intelligence with modern e-commerce. It helps farmers make data-driven decisions and sell products directly to customers.

Key Features
- Crop Prediction: Predicts best crop based on soil & environment
- Smart Product Comparison: ML-based comparison (price, features, ratings)
- Inventory Prediction: Forecasts demand
- Farmer Dashboard: Manage products
- Admin Panel: Manage users & system
- Authentication System: Secure login/signup

Technologies Used
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: PHP
- Machine Learning: Python
- API: Flask
- Database: MySQL

Project Structure
frontend/ → UI
backend/ → PHP logic
api/ → ML models
uploads/ → assets

How to Run
1. Install XAMPP/WAMP
2. Start Apache & MySQL
3. Place project in htdocs
4. Run in browser
5. Start ML API: python app.py

Objective
- Data-driven crop recommendation
- Direct selling platform
- AI-based decision support

Future Scope
